## Unicorn Studio — WebGL Effects

### Overview
- **Type:** No-code WebGL tool for websites
- **Use Case:** Animated backgrounds, interactive effects, motion graphics
- **Size:** 36kb gzipped (lightweight)

### Key Features (from research)
- 35+ WebGL effects library
- Mix/match/stack effects
- Scroll-based interactivity
- Element appearance animations
- Shader and asset optimizations
- Framer, Webflow compatible
- Any website embed

### Pricing Model
- **Freemium** (free tier available)
- **Pro:** One-time payment mentioned
- Not subscription-based (pay once, keep forever)

### Recommendation
✅ **Worth investigating** — low file size, one-time cost, works with Framer templates

### Next Steps
- Visit https://www.unicorn.studio/pricing for exact cost
- Test free tier with current SaaS template
- Create demo WebGL hero section

## Aura — AI Website Builder

### Overview
- **Type:** AI-powered landing page/site builder
- **Use Case:** Rapid template generation, inspiration
- **Audience:** Designers, template creators

### Key Features (from research)
- 65,000+ users
- Exports to HTML & Figma
- AI generates designs from prompts
- No design skills needed
- Trustpilot rated highly

### Pricing (to verify)
- Estimated ~$20/mo from business strategy doc
- May have free tier

### Use for Template Business
1. **Rapid prototyping** — Generate base designs quickly
2. **Inspiration** — AI variations for creative blocks
3. **Efficiency** — Speed up template creation workflow

### Recommendation
⚠️ **Verify pricing before committing** — Could accelerate template production if cost-effective
