# Framer Marketplace Guide
**For Pulse Graphix Template Business**

---

## Overview
- **Revenue Split**: 100% to creator (Framer takes 0%)
- **Payout**: Via Stripe Connect
- **Review Time**: Up to 14 days
- **Affiliate Income**: Earn from free templates too

---

## Submission Requirements

### Design Quality
- Original design (no copying other templates)
- Consistent typography, spacing, colors
- Fully responsive (desktop, tablet, mobile)
- Professional polish throughout
- No broken links or placeholder content

### Technical Requirements
- Built in Framer (not imported)
- Clean component structure
- Optimized images (WebP preferred)
- Smooth animations (60fps)
- SEO metadata configured

### Required Assets
- **Template Name**: Clear, descriptive
- **Short Description**: Punchy value proposition (1-2 sentences)
- **Long Description**: Detailed features, use cases
- **Screenshots**: 
  - Cover image (1920x1080)
  - 5-10 preview images
  - Mobile mockups included
- **Preview Link**: Live demo on Framer
- **Category**: SaaS, Portfolio, Agency, etc.
- **Tags**: Relevant keywords
- **Price**: $19-$199 (recommended $49-$99)

---

## Submission Process

1. **Create Framer Account** → framer.com
2. **Design Template** → Build in Framer editor
3. **Publish as Site** → Get live URL
4. **Apply for Creator** → Submit application
5. **Creator Dashboard** → Submit template
6. **Review** → Up to 14 days
7. **Publish** → Live on marketplace
